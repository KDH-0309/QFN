# QFN 프로젝트 PPT 자료 (요약본)

## 1. 프로젝트 한 줄 소개
- 양자영감 최적화(QAOA) + 현대 포트폴리오 이론(MPT)을 결합해, 조합 폭발 문제를 더 넓고 깊게 탐색하여 위험 대비 수익을 개선하는 포트폴리오 옵티마이저

## 2. 우리가 다른 점(차별점 & 강점)
- 하이브리드 최적화: MPT의 연속 최적화(평균–분산 프레이밍)에 QAOA 기반 조합 탐색을 결합 → 국지 최적 탈출, 대안 해 탐색 다양성 확보
- 실제 데이터 중심: 실시간 환율/시세 반영, 저장된 포트폴리오에도 실시간 가치·수익률 재계산
- 간결한 사용자 여정: 한 화면에서 자산 선택 → 제약 조건 → 결과 비교(동일가중치 대비), 면책 고지 및 다국어(한/영) 지원
- 배포 용이성: 정적 리소스 + 경량 서버 구조, 이미지/리포트는 `docs/backtest`에서 즉시 활용 가능

## 3. 성과 하이라이트(기존 백테스트 근거)
- 평균 추가 수익: **$933 (+23.0%)**, 초기 투자금 $10,000 기준(3개월)
- 이미지 삽입: `docs/backtest/final_portfolio_value_comparison.png`
  - 제목 예: “3개월 투자 후 포트폴리오 가치 (학습 기간별)”

## 4. MPT vs 하이브리드 비교(설명 슬라이드)
- 문제 정의: 동일한 위험 한도/제약에서 ‘MPT 단독’과 ‘MPT+QAOA’의 해 공간 탐색 방식 비교
- 핵심 차이:
  - MPT: 연속 공간에서 평균–분산 목적을 직접 최적화(해가 단일 지역으로 수렴하기 쉬움)
  - 하이브리드: QAOA가 조합/비선형 제약 하에서 다수의 유망 조합을 탐색 → 그 위에 MPT로 정교 조정
- 기대 효과: 다크호스 조합 발굴, 포트폴리오 분산효과 강화, 동일 위험 하에서 기대수익 개선 가능성 증대
- 그래프 제안(데이터 필요 시): “MPT vs 하이브리드(3개월 누적 수익/최종 가치)”
  - 자리표시 이미지 경로(교체 예정): `docs/backtest/mpt_vs_hybrid_3mo.png`
  - 현재 레포에 MPT-단독 결과 이미지가 없으므로, 동일 데이터로 MPT-only 백테스트를 돌린 후 생성 필요

## 5. 적용 가능한 비즈니스(주식 외 2~3개)
- ETF/테마 로보어드바이저: 섹터/테마 ETF 조합 최적화(자가 제약·유동성 반영) → 개인·PB 채널 상품화
- 환헤지/자금운용(Treasury): 통화·만기별 헤지 비율 조합 최적화 → 변동성/비용 최소화, 목표 듀레이션 충족
- 재고·프로모션 믹스(리테일/커머스): SKU·캠페인 조합을 예산·재고 제약 하에서 최적화 → 마진 극대화와 재고회전 개선
  - (확장) 에너지/발전 포트폴리오, 광고 예산 배분, 공급망 라우팅 등 조합·제약이 많은 영역에 일반화 가능
